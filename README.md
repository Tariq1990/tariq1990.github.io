# Portfolio Website for GitHub Pages

## Files
- `index.html`
- `style.css`
- `script.js`
- `Tariq_Jamal_Resume_Updated.pdf`

## How to publish on GitHub Pages
1. Create a new repository named `tariq1990.github.io`.
2. Upload all files from this `portfolio-website` folder.
3. Ensure `Tariq_Jamal_Resume_Updated.pdf` is in the repository root so the resume button works.
4. In GitHub: `Settings -> Pages`.
5. Source: `Deploy from a branch`, Branch: `main`, Folder: `/ (root)`.
6. Wait about 1 minute, then open `https://tariq1990.github.io`.

## Optional
- Add `CNAME` if you want a custom domain.
- Add project screenshots in an `assets/` folder and reference them in `index.html`.
